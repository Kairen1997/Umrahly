How It Works

1. User selects online payment method (Credit Card, Online Banking, E-Wallet)

2. User completes booking form with traveler details and payment information

3. System creates booking and generates payment gateway URL

4. User is redirected immediately to the external payment gateway

5. User completes payment on the external gateway

6. User returns to dashboard after successful payment

Supported Payment Methods

  • • Credit Card: Stripe Checkout
  • • Online Banking: PayPal (FPX)
  • • E-Wallet: Boost, Touch 'n Go
  • • Offline: Bank Transfer, Cash
